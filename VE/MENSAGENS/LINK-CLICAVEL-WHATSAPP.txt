Porque o link nao fica azul no WhatsApp?
======================================

O WhatsApp so torna clicavel um URL "limpo":
  https://candeias-briefing.netlify.app/

NAO coloques na mesma linha:
  - emoji antes do link (👉 https://...  <- NAO fica azul)
  - espacos antes de https
  - texto colado ao link (link:https://... sem espaco)

COMO ENVIAR
-----------
1) Abre: 05b-whatsapp-colar-direto.txt
2) Ctrl+A, Ctrl+C
3) Cola no WhatsApp e ENVIA a mensagem
4) Depois de enviar, o link fica azul para quem recebe

Nota: no campo de escrever as vezes so fica azul DEPOIS de enviar.
      Para ti tambem deve ficar azul na conversa enviada.

Se ainda nao mudaste o nome no Netlify, usa o link que tens la:
  https://exquisite-cajeta-30c006.netlify.app/
(mesmas regras — so o URL numa linha so)
